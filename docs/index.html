Nine modular mini-robots. No leader. Gait encoded as graph topology, signed edges tune phase from gallop to trot, and fault tolerance scales directly with graph degree. Worst-case phase error drops 3x vs leader-follower. https://arxiv.org/abs/2607.07281v1
